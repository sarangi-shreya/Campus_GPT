# CampusGPT

CampusGPT is an AI-powered smart campus assistant designed to help students with academic and campus-related activities through an interactive and user-friendly platform. The project aims to simplify student life by providing instant assistance for academic queries, schedules, campus information, and student support services.

## Features

* AI-based chatbot for student queries
* Academic and subject-related assistance
* Timetable and schedule support
* Campus information and navigation help
* Event and announcement updates
* Personalized student interaction
* User-friendly interface
